Sobre o Projeto

Este projeto foi desenvolvido como parte da atividade prática "Desenvolvimento de Interfaces Web". O objetivo é criar um site dinâmico e responsivo utilizando Bootstrap para o layout, JavaScript para carregar dados de uma estrutura JSON, e CSS personalizado para a estilização, demonstrando os conceitos de entidade principal e secundária.

Aqui, exploramos o universo dos times de futebol, exibindo uma lista de clubes com uma página de detalhes para cada um, mostrando sua história, títulos e galeria de fotos.
